Sell Devllo Events Tickets with WooCommerce

== Description ==
Integrates Devllo Events with WooCommerce to sell Devllo Event Tickets as a product.



== Installation ==
You can install directly from the Plugin Repository using the Plugins system in WordPress.

= MANUAL INSTALLATION =
Upload devllo-events.php to the /wp-content/plugins/ directory
Activate the plugin through the ‘Plugins’ menu in WordPress


== Frequently Asked Questions ==
= Assigning a product to an event =
- Add a new WooCommerce product and set the price you will charging for the event
- Check "Set as Event Ticket Product"
- Create a new event or edit a current event
- Under "Select Event Ticket", select the event product you have created.
